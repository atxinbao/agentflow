## Runtime Boundary

Local runtime facts are generated by AgentFlow and are not source archive
authority. Do not commit these local runtime records as release source:

- `.agentflow/runs/**`
- `.agentflow/tmp/**`
- `.agentflow/tasks/**`
- `.agentflow/index.sqlite`
- `.agentflow/index.sqlite-*`

Stable agent roles and operating rules belong in tracked documentation. Local
materialized agent manuals under `.agentflow/define/agent/**` are runtime
outputs and must not replace this source entry.
