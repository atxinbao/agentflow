## 授权边界

- `ROADMAP.md` 只定义顺序。
- `construction-plan.md` 只定义候选施工包。
- `mvp-spec.md` 只定义格式和行为。
- 只有明确 issue contract 才能授权实现。
- Human 未明确说“开始实现”前，不创建代码骨架。
