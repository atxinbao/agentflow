## Linear -> AgentFlow

| Linear 概念 | AgentFlow 采用 | 不采用 |
| --- | --- | --- |
| Workspace / Team | `LocalWorkspace` / `LocalProject` 的本地边界 | SaaS 团队和权限体系 |
| Issue / Workflow | `IssueContract` + `ExecutionRun` | 完整 PM 状态机 |
| Project / Milestone | project 聚合 roadmap、issue、evidence | 企业排期系统 |
| Views | `SavedView` filter | 复制事实源 |
| Project Updates | `ProjectUpdate` 从 evidence 派生 | 周报平台 |
| Agents | `AgentProfile` + run 状态可观察 | 团队成员权限模型 |
| API / SDK / Webhooks | future integration boundary | MVP 远程 GraphQL/webhook |

## 当前蓝图

```text

-> GoalCompiler
```

设计主视图：Project、Map、Issues、Run、Evidence、Review。Views 只是 saved filters，不是第 7 个主视图。

## 明确边界

- 不做 Linear 替代品。
- 不在 MVP 做 SaaS workspace/team、cycles、initiatives、enterprise roadmap。
- 不在 MVP 做远程 GraphQL API、webhook 平台或云同步。
- Roadmap / candidate issue 不等于执行授权。
