# Software Dev Examples

更新日期：2026-06-26
执行者：Codex

## Purpose

本目录后续保存 Software Dev AgentFlow Product 的标准场景样例。

## Required Example Types

| 示例 | 应覆盖 |
| --- | --- |
| feature request | Intent -> Spec Bundle -> Plan -> Issue -> Delivery |
| bug fix | failure evidence -> fix issue -> verification -> decision |
| release | release gate -> provenance -> certification -> delivery |
| audit | independent audit request -> finding -> report |
| design-only | UI / product design output without Build Agent authorization |

## Rule

示例必须服务行业合同，不能变成新的事实源。真实执行仍然以 confirmed Spec Bundle 和 `.agentflow/spec/**` 为准。
